# 🌟 Skillora

Skillora is an AI-powered daily skill learning system that helps users master any skill through structured roadmaps, daily missions, and AI coaching.

---

## 🚀 Features

- 🧠 AI-generated learning roadmaps
- 🔥 Daily 1-task focus system
- 🤖 AI learning coach (Gemini-powered)
- 📈 Progress tracking system
- ⚡ Minimal distraction learning flow

---

## 🌐 Live Project

Coming soon (deployed on Vercel)

---

## 🧩 Tech Stack

- HTML / CSS / JavaScript
- Vercel Serverless Functions
- Google Gemini API

---

## 🎯 Purpose

Skillora is built to solve one problem:

> People don’t fail because they lack resources — they fail because they lack structure.

Skillora provides a structured, AI-driven daily learning system.

---

## ⚙️ How it works

1. User selects a skill
2. AI generates a 7-day roadmap
3. User completes 1 task per day
4. AI coach guides progress
